## How to serve
```Bash
sudo apt-get install python-software-properties
curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
sudo apt-get install nodejs

cd my-vue-practice

rm -r node_modules
npm install
npm rebuild node-sass
npm install -g gulp
```

```Bash gulp watch```

or

```Bash gulp serve```
